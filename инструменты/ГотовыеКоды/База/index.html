Детектор текущего дня
JS:
function isCurrentDate(currentMonth, currentDay) {
  const today = new Date();
  const currentMonthZeroBased = currentMonth - 1;
  return (
    today.getMonth() === currentMonthZeroBased && 
    today.getDate() === currentDay
  );
}
Пример использования:
isCurrentDate(1, 1); // true, если сегодня 1 января
isCurrentDate(12, 31); // true, если сегодня 31 декабря
