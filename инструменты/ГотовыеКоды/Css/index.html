Клевый Контейнер
Ставит элементы в несколько рядов под друг друга если ширина экрана меньше или равна 768 пикселям, если ширина больше то элементы можно листать.
CSS:
div.control-cont {
@media (max-width: 768px) {
margin: 20px;
padding: 40px;
display: flex;
flex-wrap: wrap;
justify-content: center;
padding: 20px;
}
@media (min-width: 769px) { 
overflow: auto; 
width: 100%; 
height: 300px;
display: flex;
overflow-x: auto;
gap: 25px;
}
}
